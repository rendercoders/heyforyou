排序算法有很多，桶算法、冒泡排序、快速排序等等。
<!-- more -->

```
// 例如100分的试卷 从高到低排序
$numbers = [12,45,56,46,73,56,33];

// 一百零一个桶初始化为0
$barrel = [];
for ($i=100; $i >= 0; $i--) {
    $barrel[$i] = 0;
}

// 有n个同学
$classmates = count($numbers);

// 计算某个桶的分数出现的次数
for ($i=$classmates; $i >= 0; $i--) {
    $barrel[$numbers[$i]]++;
}

// 依次判断是否存在
for ($i=100; $i >= 0; $i--) {
    // 有多少次该分数打印多少次
    for ($k=$barrel[$i]; $k >= 1; $k--) {
        echo $i;
        echo "<br/>";
    }
}

// 打印出来的结果为
73
56
56
46
45
33
12
```
