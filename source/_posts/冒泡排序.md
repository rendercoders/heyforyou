详细理解先空着，先给出例子。等理解深了再来补充。

```
$numbers = [12,45,56,46,73,56,33];
$n = count($numbers); // 获取个数
for ($i=0; $i < $n-1; $i++) {  // n个数排序，只需要进行n-1次
    for ($j=0; $j < $n-$i; $j++) {
        if ($numbers[$j] < $numbers[$j+1]) { //比较大小
            $t = $numbers[$j];
            $numbers[$j] = $numbers[$j+1];
            $numbers[$j+1] = $t;
        }
    }
}
```
